Про меня

Меня зовут Алина

Привет! Я Алина, backend разработчик. И здесь я разместила информацию о себе и своих проектах в программировании.  

В прошлом автотестировщик с 3-летним стажем. Мне нравится работать с кодом, разбираться как построен процесс, его логику. Когда сидишь над задачей часами или днями, и потом получается и работает — это офигенное чувство. Поедая «большого цифрового слона» по кусочкам, все больше понимаю, насколько он «вкусный»).

Разработка приложений

Пишу чистый, поддерживаемый код на C#, соблюдая принципы SOLID и паттерны проектирования.

Оптимизация и производительность

Оптимизирую код и архитектуру приложений для повышения их скорости и масштабируемости.

Интеграция и API

Работаю с REST API, создаю и интегрирую клиентские и серверные решения.

Тестирование и отладка

Разрабатываю автотесты, провожу отладку и исправление ошибок, обеспечивая стабильность и качество продукта.

Стек технологий

C#

Основной язык программирования. Пишу масштабируемый и поддерживаемый код, используя LINQ, async/await и современные подходы.

ASP.NET Core

Фреймворк для создания мощных и быстрых веб-приложений. Использую для разработки API, работы с серверной частью и интеграции клиентских решений.

Entity Framework

ORM для работы с базами данных. Помогает эффективно проектировать, управлять и оптимизировать взаимодействие приложения с данными.

Git

Система контроля версий. Работая с Git, веду историю изменений, участвую в командной разработке и управляю ветками для стабильной интеграции.

Postman

Инструмент для тестирования API. Проверяю запросы, отлаживаю взаимодействие клиент-сервер, автоматизирую тесты для стабильной работы приложений.

Helm

Применяю для управления и развертывания приложений в Kubernetes, упрощая конфигурацию микросервисов.

CI/CD

Интеграция и автоматизация процессов разработки. Настраиваю пайплайны для тестирования и развертывания, что минимизирует ошибки и ускоряет выпуск продукта.
